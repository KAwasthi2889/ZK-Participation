# ZK-Participation Smart Contracts

Confidential smart contracts for the ZK-Participation hackathon project.

## Quick Start

### Prerequisites

- Node.js 22+
- [Compact CLI](https://docs.midnight.network) — installed via GitHub release (not npm)

### Install Compact CLI

```bash
curl --proto '=https' --tlsv1.2 -LsSf https://github.com/midnightntwrk/compact/releases/latest/download/compact-installer.sh | sh
export PATH="$HOME/.local/bin:$PATH"
compact update
compact --version
```

### Compile the Contract

```bash
cd contracts
compact compile src/ClinicalTrialCredential.compact ./managed
```

## Smart Contract: ClinicalTrialCredential

### Circuits (8 total)

| Circuit | Auth | Side Effects | Returns |
|---------|------|--------------|---------|
| `issueCredential` | Sponsor admin only | Inserts into `credentials` Map | `[]` |
| `revokeCredential` | Sponsor admin only | Inserts into `revokedCredentials` Set | `[]` |
| `generateProof` | Patient (witness) | None | `Boolean` |
| `verifyProof` | Verifier (public) | None | `Boolean` |
| `hasActiveCredential` | Any | None (read-only) | `Boolean` |
| `isRevoked` | Any | None (read-only) | `Boolean` |
| `getCredentialHash` | Any | None (read-only) | `Bytes<32>` |
| `getCredentialCount` | Any | None (read-only) | `Uint<64>` |

### Privacy Model

| What | Protection |
|------|------------|
| Patient identity | **Never on-chain** — derived from witness secret |
| Trial details | **Never on-chain** — hidden inside `persistentCommit` |
| Completion status | **Never on-chain** — hidden inside `persistentCommit` |
| Issue date | **Never on-chain** — hidden inside `persistentCommit` |
| Commitment hash | Public (hides data via random salt) |
| Sponsor public keys | Public (needed for authorization) |
| Boolean proof result | Public (the entire purpose) |

## Frontend Integration

The `src/blockchain.ts` file provides **drop-in replacements** for the 5 mocked blockchain functions in `frontend/src/app/App.tsx`:

```typescript
// Before (mocked):
import { connectWallet, issueCredential, revokeCredential, generateProof, verifyProof } from './mockBlockchain';

// After (real contract SDK):
import { connectWallet, issueCredential, revokeCredential, generateProof, verifyProof } from '../../contracts/src/blockchain';
```

Function signatures are **100% compatible** — no UI changes needed.

### Two Modes

**Hackathon Demo Mode (no devnet):**
- Uses browser `localStorage` for private state
- Simulates contract calls client-side
- Perfect for the 2-minute hackathon demo

**Production Mode (with devnet):**
- Same function signatures
- Replace `localStorage` with real contract SDK calls
- Connect to deployed contract address

## Data Flow

```
Sponsor Dashboard
  │
  ▼
issueCredential(participantPk, trialId, completionStatus, issueDate)
  │
  ├─ Auth: deriveSponsorPublicKey(getUserSecret()) == sponsorAdminPk
  ├─ Build: ClinicalTrialData struct
  ├─ Commit: persistentCommit(data, salt) → hash
  └─ Store: credentials.insert(participantPk, commitmentHash)
  │
  ▼
Midnight Network (confidential state)
  │
  ▼
Patient Dashboard
  │
  ▼
generateProof()
  │
  ├─ Derive: patientPk = derivePatientPublicKey(getUserSecret())
  ├─ Lookup: credentials[patientPk] → storedCommitment
  ├─ Recompute: persistentCommit(data, salt) from witness
  ├─ Verify: recomputed == storedCommitment ✓
  ├─ Check: !revokedCredentials.contains(storedCommitment) ✓
  └─ Return: true + proof JSON
  │
  ▼
Verifier Dashboard
  │
  ▼
verifyProof(patientPk, claimedCommitment)
  │
  ├─ Check: credentials[patientPk] exists
  ├─ Check: storedCommitment == claimedCommitment
  ├─ Check: !revokedCredentials.contains(storedCommitment)
  └─ Return: "✅ Valid Clinical Trial Participant"
```

## Development Notes

### Patterns Used

| Source | Adaptation |
|--------|------------|
| `SelectiveDisclosure.compact` | `persistentCommit<ClinicalTrialData>` + salt hiding |
| `midnight-rwa` (PassportIdentity) | Witness-derived identity via `derivePatientPublicKey` / `deriveSponsorPublicKey` |
| `zkloan-credit-scorer` | Admin-only issuance/revocation with `assert(callerPk == sponsorAdminPk)` |
| `AccessControl.compact` | Role-based authorization patterns |

### Compiler Fixes Applied

| Error | Fix |
|-------|-----|
| `operation = undefined for Counter` | Removed `credentialCounter = default<Counter>` — `Counter` auto-initializes to 0 |
| `expected structure type, received Uint<64>` | Changed `credentialCounter.value` → `credentialCounter.read()` |
