# Frontend Developer (Person 1)

## Your Mission
Build the web application that users interact with.

You don't need to understand zero-knowledge proofs or how Midnight works internally. Think of the blockchain like a backend API that exposes a few functions.

You'll build three simple screens:

### 1. Sponsor Dashboard
A researcher/hospital can:
- Connect their wallet
- Enter a participant's wallet address
- Click **Issue Credential**

### 2. Patient Dashboard
A participant can:
- Connect their wallet
- See whether they have a credential
- Click **Generate Proof**

### 3. Verifier Page
A verifier can:
- Paste or receive a proof
- Click **Verify**
- See: ✅ Valid Clinical Trial Participant or ❌ Invalid Proof

Your job is to make the app intuitive, responsive, and demo-friendly. Whenever a button is clicked, it should call a function provided by the blockchain teammate.

## 30-Hour Timeline
- **Phase 1: Setup & Architecture (Hours 0 - 5)**: Initialize React/Next.js repository, set up routing, build basic UI components.
- **Phase 2: Core Development (Hours 5 - 15)**: Implement Midnight Wallet connection, build views for Sponsor, Patient, and Verifier.
- **Phase 3: Integration (Hours 15 - 22)**: Refine UI/UX based on integration feedback, handle loading states and errors.
- **Phase 4: Polish (Hours 22 - 27)**: Final CSS/styling polish, ensure the app is responsive and presentable.
- **Phase 5: Demo Prep & Buffer (Hours 27 - 30)**: End-to-end walkthrough, record demo video, buffer for unexpected bugs.
